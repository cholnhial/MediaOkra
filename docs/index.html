About

Have you ever wanted to just use Google Assistant to pause or stop a movie or music playing on your desktop or laptop. That's what MediaOkra was created for. MediaOkra is a Google Assistant Action that comes with a client which runs on Mac and Windows. The client receives commands from the action via a secret code that you get given when you sign in to the Action. So With MediaOkra you can now tell all your dumb devices to shut up. It works best with Google Home or Home Mini.

Getting Started

Windows

Step 1.

Download the Binary

Click to MediaOkra 64-bit Executable

Step 2.

Unblock executable

Go to the location of the downloaded file and right click it. Go to properties

At the bottom you should see a button that says "Ublock" click on it then hit Apply and Okay.

Step 3.

Start the executable by clicking on it.

Step 4.

Start MediaOkra on your Google home or phone by saying "Okay Google, Talk to Media Okra". Then after the welcome message say "sign in", Media Okra will generate a code for you enter this code into the client on your PC and click Start Listening. You can always ask MediaOkra for the code by saying "my code"

Now you are free to say commands like "pause", "play" and "mute" to MediaOkra.

Mac OSX

Step 1.

Download the Jar

Click to Download Jar

Step 2.

Click on the Jar once downloaded. You must make sure Java is Installed. At first it won't start because it was downloaded from the internet. So go to your System Preferences from the launcher. Then go to Security & Privacy. Then to the General Tab and look for the app name below to ublock it.

Step 3.

Once unblocked you can click on the jar again and it should start up.

Start MediaOkra on your Google home or phone by saying "Okay Google, Talk to Media Okra". Then after the welcome message say "sign in", Media Okra will generate a code for you enter this code into the client on your PC and click Start Listening. You can always ask MediaOkra for the code by saying "my code"

Now you are free to say commands like "pause", "play" and "mute" to MediaOkra.
